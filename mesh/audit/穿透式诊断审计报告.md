## 6. 升级优化小结

1. **并发安全**：单锁层（fcntl EX）覆盖 load→modify→save，进程内+跨进程双重安全。
2. **持久化原子性**：temp+rename+fsync，崩溃零损坏。
3. **健壮性**：输入校验 + 幂等去重，防注入与膨胀。
4. **可观测**：`/metrics` 实时暴露认知健康。
5. **契约一致**：feed 与 server 共用同一把锁，杜绝跨进程竞态。
