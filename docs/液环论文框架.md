# 液环（Liquid Loop）论文框架

> **North-Star 公理（一切代码与论文围绕它校验）：**
>
> **Liquid Loop is not a memory storage mechanism; it is a self-regulating memory state evolution mechanism.**
>
> （液环不是一种记忆存储机制，而是一种自调节的记忆状态演化机制。）
>
> 衍生定位语：Self-Regulating Persistent Memory Dynamics for Agent Systems
> （面向智能体系统的自调节持久记忆动力学）
>
> 注：不主打"AI 意识 / 认知层"叙事；以可验证的动力学机制切入，工程与科研接受度更高。

## 1. 问题陈述（Motivation）

大语言模型智能体的长期记忆面临三类根本问题：

1. **来源可信度缺失**：记忆"为什么可信"无据可查 → 液环以**链式审计（AuditChain, SHA256）**解决。
2. **群体幻觉固化**：单纯"≥N 条一致即真"会把集体错误固化为记忆 → 液环以**反证轨（Contradiction Track）**打破。
3. **静态存储谬误**：记忆被当作不变对象，忽视衰减/强化/冲突的动态过程 → 液环以**时间动力学 `step(dt)`** 建模为过程。

## 2. 核心机制（Mechanism）

### 2.0 边界：液环不是什么（防止退化为"智能记忆管理器"）

液环的独特性必须守住一条本体论线：

**记忆状态本身是一个演化对象，而不是被管理的数据对象。**

市场已有的"智能记忆管理器"范式——不应成为液环的归宿：
- **记忆评分（memory scoring）**：只打分、不演化；
- **自动删除（auto-delete / 过期清理）**：把衰减外置为人工规则，而非状态内生动力学；
- **权重调整（weight tuning）**：把动力学降维成静态参数。

这些把记忆当**被管理的数据对象**。液环要保护的是 **memory homeostasis（记忆稳态）**：记忆在变化环境中经
`输入 → 吸收 → 凝聚 → 稳定 → 衰减 → 重构` 自行维持一致性，而非被外部调度规则指挥。

这正是"液态"设想的承诺：静态 memory = 写入→保存→调用；液环 = 输入→吸收→凝聚→稳定→衰减→重构 的动态系统。

### 2.1 状态—输入—聚合—稳定性—追溯 闭环

```
Input Evidence
   ↓
Memory State (Anchor → Evidence → Memory)
   ↓
LEI Evaluation (Liquid Entropy Index, 八维加权)
   ↓
Decay / Reinforcement (step(dt))
   ↓
New Memory State ——→ AuditChain (SHA256 链式哈希)
```

### 2.2 反证轨（v0.8）

- `Evidence.relation ∈ {support, contradiction}`，`target_memory_id` 显式指向被反驳记忆。
- 记忆稳定性：`stability = support / (support + 2·contradiction + 1)`。
- 语义：**一致增加稳定性，冲突降低稳定性**——而非"一致产生真"。

### 2.3 时间动力学（v0.8）

- 显式演化：`M(t+1) = M(t) + reinforcement − decay − contradiction_penalty`。
- 证据权重随 `(1−decay_rate)^dt` 衰减；记忆在获得新 support 时恢复固有稳定性，否则时间衰减。
- 记忆是**过程**而非对象。

### 2.4 多主体共识（MESH v2）

- `private` 轨（同 agent ≥2 一致）+ `consensus` 轨（≥2 distinct agent 一致）。
- **零向量硬约束**：一致性判定走结构化精确相等 + 审计链哈希，绝不引入 embedding / 相似度。

## 3. 可验证实验（Empirical）

| 实验 | 假设 | 结果 |

（详见 `examples/experiments/`，运行 `python3 examples/experiments/run_all.py`）

## 4. 与相关工作对比

- **向量记忆库（RAG/MemGPT 类）**：依赖 embedding 相似度，语义近≠认知一致；液环以结构化判定规避。
- **静态知识图谱**：无衰减/强化动力学；液环引入显式时间演化。
- **神经图灵机 / 微分记忆**：连续可微但不可解释；液环走离散可审计闭环。
