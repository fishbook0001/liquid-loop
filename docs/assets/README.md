## 使用规范

- **主视觉以上述原图为准**，Vera 生成的 SVG 草稿（`output/liquid-loop/` 下）仅为备选，**不进仓库 `docs/brand/`、不替代主视觉**。
- 引用时统一用本目录文件，不要使用 `output/liquid-loop/originals/` 下的 clipboard 原名。
- 随 **v0.9.0** 发版一并提交推送（不单独发布）。

## 配套品牌规范

- 定位公理（North-Star）：`Liquid Loop is not a memory storage mechanism; it is a self-regulating memory state evolution mechanism.`
- 色板（以原图作者飞哥标注为准）：主蓝 `#0A22CC`、青绿 `#2DD4BF`、状态绿/黄/红 `#22C55E`/`#EAB308`/`#EF4444`。
- 详见仓库根 `README.md` 的「定位」章节与 `docs/液环论文框架.md`。
