# 液环抖音引流视频 — 分镜脚本

## 目标
- 平台：抖音竖屏 9:16（1080×1920）
- 时长：60-75 秒
- 目标受众：AI 开发者、Agent 爱好者
- 核心引流点：GitHub star + PyPI install

## 分镜（7 场景）

### Scene 1: Hook（0-5s）
- 画面：深色背景 + 大字 "你的 Agent 还在用 LLM 管记忆？"
- 配音：你的 Agent 还在用大模型管理记忆？
- 动效：文字从底部弹入

### Scene 2: 痛点（5-12s）
- 画面：三个痛点卡片依次从右滑入
  - "向量检索 → 需要外部评分"
  - "LLM 摘要 → 需要外部压缩"
  - "图数据库 → 需要 LLM 诊断"
- 配音：向量检索要外部评分，LLM 摘要要外部压缩，图数据库要 LLM 诊断。全都要大模型介入。

### Scene 3: 转折（12-18s）
- 画面：大字 "如果记忆能自组织呢？"
- 配音：如果记忆能自组织呢？不需要任何外部干预。

### Scene 4: 核心原理（18-35s）
- 画面：三层结构（锚点→证据→结晶），粒子飘入 + 结晶闪光
- 配音：液环的原理很简单。注入证据，证据自动聚向锚点。两条以上一致的证据，自动结晶成记忆。不需要排序，不需要检索。熵值天然反映认知健康度。

### Scene 5: 性能数据（35-45s）
- 画面：三个数据卡片弹入
  - "73K 写入/秒"
  - "0.014ms 单条延迟"
  - "零 LLM 依赖"
- 配音：每秒七万三千次写入，单条延迟零点零一四毫秒。零LLM依赖，纯Python。

### Scene 6: 代码展示（45-53s）
- 画面：五行代码 + 注释
- 配音：五行代码，自组织记忆。

### Scene 7: CTA（53-60s）
- 画面：GitHub + PyPI 信息 + 关注提示
- 配音：GitHub 搜液环，star 一下。关注我，后面还有更多 Agent 架构干货。
